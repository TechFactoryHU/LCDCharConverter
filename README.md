LiquidCrystal_I2C extension
=============
This is an extension class for LiquidCrystal_I2C (by fdebrabander, https://github.com/fdebrabander/Arduino-LiquidCrystal-I2C-library)
This class can be used to display some UTF-8 characters on a standard LiquidCrystal LCD driven by I2C (LCD-I2C-IF) 
Hungarian letters supported currently (áÁ éÉ íÍ õÕ úÚ óÓ üÜ öÖ), but feel free to extend the charmap.

Character editing
=============
Useful online tool for char editing: https://maxpromer.github.io/LCD-Character-Creator/
